# Screenshots Evidence Guide

## Naming convention
Use one screenshot per flag minimum.

Examples
- `flag01_initial_endpoint.png`
- `flag03_powershell_payrollsupporttool.png`
- `flag07_outbound_connectivity_test.png`
- `flag22_final_outbound_remote_ip.png`

## Minimum checklist
Flag 01 through Flag 22: at least one screenshot each showing the query results table and the key value that answers the flag.
